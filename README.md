#### 精读圣经
精读圣经是一款功能强大，资源丰富，可完全离线使用的圣经工具。  
电脑版有专门的投影模块，投影非常赞！除此之外还含有模块制作功能，可用于制作译本、注释、灵修、辞典及读经计划。制作的资料可以上传服务器，以「口令」的形式提供下载，也可以使用「数据传输」功能分发到其他设备。
